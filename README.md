# Database demo

This repository is a demo for database subject , Chulalongkorn University

## Requirement
* Python3.6
* MySQL

## Quick start
* run `pip3 install -r requirements.txt` to install module
* run `python3 flaskr/main.py` to run the application in PORT 8080
