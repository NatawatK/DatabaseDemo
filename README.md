# Database demo

This repository is a demo for database subject , Chulalongkorn University

## Requirement
* Nodejs
* MySQL

## Quick start
* Clone this repository and run `npm install` or `yarn` to install packages
* Run `npm run watch` to compile , run and watch this workspace
